사용 설명서

📝 기록 추가하기

  • 일명: 어떤 일을 했는지 입력합니다
  • 날짜: 그 일을 한 날짜를 선택합니다
  • 주기 (선택): 이상적인 반복 주기를 설정할 수 있습니다
  • 추가 버튼: 클릭하면 기록이 저장됩니다

🔄 정렬 기능

  • 지난일 기준 ▲/▼: 마지막으로 한 일을 기준으로 정렬합니다
  • 남은일자 기준 ▲/▼: 주기가 설정된 항목을 남은 일수로 정렬합니다
  • ▲: 작은 순서대로 정렬 (오름차순)
  • ▼: 큰 순서대로 정렬 (내림차순)

🔧 항목 관리

  • 리프레시: 항목의 날짜를 오늘로 업데이트합니다
  • 삭제: 항목을 완전히 삭제합니다
  • 히스토리: 클릭하면 이전 기록들을 확인할 수 있습니다

📊 진행도 표시

주기가 설정된 항목은 배경 그라데이션으로 진행도를 시각화합니다.

  • 초록색: 아직 시간이 남음
  • 빨간색: 주기를 초과함 (⚠ 경고)

💾 백업 / 복원

데이터는 이 브라우저의 localStorage 에만 저장됩니다. 다른 기기로 옮기거나 안전하게 보관하려면 JSON 으로 내보내세요.

가져오기를 하면 현재 데이터를 덮어씁니다.
